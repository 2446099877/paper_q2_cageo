# C&G Data Availability Statement

更新时间：`2026-03-23`

## 建议提交文本

The datasets analyzed during the current study are publicly available. AID is available from the original dataset source described by Xia et al. (2017), and NWPU-RESISC45 is available from the original dataset source described by Cheng et al. (2017). The project repository provides split manifests, experiment configurations, and reproduction scripts, but does not redistribute the raw image files.
